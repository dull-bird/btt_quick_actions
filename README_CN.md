# BetterTouchTool QuickActions 预设使用指南

*[Read this in English](README.md)*

这份文档介绍了当前 [BetterTouchTool (BTT)](https://folivora.ai/) 预设中包含的快捷手势和快捷键逻辑。你可以根据这些手势介绍快速上手，或者将它分享给其他人作为使用参考。

## 🚀 快速开始

1. 下载 `QuickActions.bttpreset` 文件。
2. 打开 **BetterTouchTool**。
3. 点击右上角的 **Manage Presets**（管理预设）。
4. 点击 **Import**（导入）并选择下载的文件。
5. 确认预设已启用且“Global”已被勾选。

---

## 🌐 全局手势

全局手势适用于几乎所有日常应用及使用场景。为了方便记忆与反馈，部分手势在触发时屏幕上会显示专属的 Emoji 提示（HUD）。

### 🤟 三指操作
主要用于**窗口管理**和**标签页切换**。

| 触发操作 | 执行动作 / 快捷键 | 屏幕提示 (HUD) $\quad$ |
| :--- | :--- | :--- |
| **三指向下轻扫** | `Command + W` (关闭当前窗口/标签页) | 无 |
| **三指向上轻扫** | `Command + Tab` (应用界面切换) | 无 |
| **三指向左轻扫** | `Ctrl + Tab` (切换至右侧/下一个标签页) | 无 |
| **三指向右轻扫** | `Ctrl + Shift + Tab` (切换至左侧/上一个标签页) | 无 |
| **三指点击** | 将当前窗口最大化并移至 **下一个显示器** | ✈️ |
| **三指连续轻点两次** | `Ctrl + Cmd + D` (在光标处呼出 macOS 词典定义) | 无 |

**搭配修饰键的高级三指手势**：
> [!TIP]
> **设计逻辑**：修饰键与 macOS 窗口左上角的物理按钮进行了直观映射，方便记忆：
> - **`Fn`** ⟷ 🔴 (关闭窗口)
> - **`Control (⌃)`** ⟷ 🟡 (最小化窗口)
> - **`Option (⌥)`** ⟷ 🟢 (全屏模式)

* 🔴 按住 `Fn` + **三指向下轻扫**：关闭光标下方的窗口
* 🟡 按住 `Control (⌃)` + **三指向上轻扫**：打开「任务控制 (Mission Control)」
* 🟡 按住 `Control (⌃)` + **三指向下轻扫**：最小化当前悬浮在光标下方 窗口
* 🟢 按住 `Option (⌥)` + **三指向上轻扫**：进入应用全屏模式（前提是应用支持）

### 🖖 四指操作
主要用于**应用级控制**和**系统工具**。

> [!WARNING]
> **安全警告**：四指 **向下轻扫** 映射为 `Command + Q` (完全退出应用)。请谨慎使用以免误触导致数据丢失。如果你觉得该手势容易产生冲突，可以在 BTT 设置中手动关闭它。


| 触发操作 | 执行动作 / 快捷键 | 屏幕提示 (HUD) |
| :--- | :--- | :--- |
| **四指向下轻扫** | `Command + Q` (完全退出当前应用) | 无 |
| **连续轻点两次** | 开启 macOS 原生截屏/录屏工具区 (Mojave及以上支持) | 无 |

### 👆 单指操作
* 按住 `Option (⌥)` + **单指轻点触控板底部中心**：触发 `Command + M` (最小化当前窗口)

## ⌨️ 全局快捷键

预设内部整合了一些深度的快捷键配置（采用特定的修饰组合键）。由于键盘上的快捷键设定较为复杂，你可以直接了解它们所能赋予的**核心能力面板**：

* **📏 窗口分屏与定位**：包含“最大化窗口至左偏 / 右偏”，帮你快速完成左右工作区的切分。
* **↔️ 显示器跨越**：直接使用快捷键将当前窗口“最大化至下一个显示器”。
* **🎯 弹性的窗口缩放**：包含名为“变得更大”和“变得更小”的自定义窗口调整操作，且缩放完成后会**自动将窗口居中对齐**。
* **✖️ 高效闭窗**：通过快捷键直接“关闭光标下方的窗口”（在 Mission Control 中依然有效）。
* **💡 屏幕控制**：快捷键控制屏幕“亮度上升”与“亮度下降”。

## 🖱️ 常规鼠标适配

部分配置照顾了外接普通鼠标的用户：
* **鼠标侧键 (Button 4)**：直接**最大化**当前窗口。
* **鼠标按键 (Button 3)**：直接**最小化**光标浮停处的窗口。

## 📦 针对特定应用的优化

对于部分特殊应用，预设做出了专门适配：

### 1. Safari 浏览器
* 为 Safari 绑定了通过**触控板左右滚动**来模拟按压 `Ctrl + Tab` 与 `Ctrl + Shift + Tab`，用以快速切换标签页。
* *(注：该功能在当前的预设中被临时设为“已禁用”状态，若有需要可在 BTT 内手动取消禁用。)*

### 2. PDF Expert (PDF阅读器)
* 为其内部逻辑强化了前后翻页/导航：
  * 按下 `[` 键将触发 `Command + [`
  * 按下 `]` 键将触发 `Command + ]`
  * 特点即：在软件内简化快捷键按压，仅通过方括号就能执行快捷后退与前进。

## ⚙️ 高级全局设置与调优




预设内部包含了深入的系统级调优 (BTTGeneralSettings)，这不仅提升了手势的触发成功率，还优化了 macOS 的原生体验：

* **🧲 增强的窗口吸附 (Window Snapping)**：
  * **全方位吸附**：开启了屏幕顶部、底部及四个角落的窗口快速吸附与扩展。
  * **视觉与体验**：配置了顺滑的吸附预览动画与高亮颜色，并启用了圆角边框 (`CornerRadius`) 与增强型拖拽手感（边缘触发灵敏度优化）。
  * **内存优化**：开启了吸附区域的内存节省模式 (`MemorySaver: True`) 与针对“台前调度 (Stage Manager)”的边缘间隙微调。

* **🖐️ 触控板防误触与压感 (Trackpad & Force Touch Tuning)**：
  * **力度控制**：针对“重按 (Force Click)”与“轻按 (Normal Click)”分别配置了精确的压力阈值（重按的触发下限显著增加，降低误触可能），并匹配细致的 Haptic 震动反馈机制。
  * **智能防误触**：默认启用了“手掌识别 (`PalmRecognition`)”以及“允许大拇指放在触控板边缘 (`HandleThumbsRestingOnTopEdgeAndCorners`)”，在双手复杂手势时不容易产生指控冲突。
  * **灵敏度微调**：对三指及四指轻扫 (`Swipe`) 设置了自定义的灵敏阈值，同时针对双击延迟 (`DoubleTapDelay`) 进行精准微调。

* **💻 异形屏与扩展界面适配 (Notch & Touch Bar)**：
  * 针对拥有刘海屏 (Notch) 或 Touch Bar 的 Mac，配置了状态栏图标的隐藏与展示逻辑。
  * 对于鼠标重度使用者，支持过滤 Magic Mouse 的边缘误触。
